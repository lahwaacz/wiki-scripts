ws.checkers.CheckerBase.localize_flag(wikicode, node, template_name)¶

If a node in wikicode is followed by a template with the same base name as template_name, this function changes the adjacent template’s name to template_name.

Parameters

  • wikicode – a mwparserfromhell.wikicode.Wikicode object

  • node – a mwparserfromhell.nodes.Node object

  • template_name (str) – the name of the template flag, potentially including a language name
